# DXRP

**DXRP** is a roleplaying game built on **s&box** with a global economy, and session-based gameplay. In this game, players engage in a dynamic world where currency is the only global factor, and each session creates an unique roleplaying experience.

This repository is the public counterpart to the DXRP repository and includes the SDK, Server Image, and issue/project tracking.

- [s&box Server Container](https://hub.docker.com/r/dxura/sbox)

## Features

- **Global Economy**: A shared in-game currency system across all sessions.
- **Roleplay Gameplay**: Mimics the core mechanics of DarkRP while providing a unique, session-only experience.
- **Admin Interface**: A web application (admins only) for server management and control.

## Gameplay

- Players take on different roles and interact.
- A dynamic economy drives interactions, and currency is shared globally.
- Session-based gameplay ensures unique experiences each time a player joins.
